the import of the code review

  • 提高代码质量
  • 减少后续的返工造成的浪费
  • 从他人的代码中学习更多的知识
  • 对整个项目工程有更加深入的理解

过程：

  • 系统全貌
  • 模块级别
  • 类、函数内部逻辑
  • 细节

​

标准：

  • 高效（Fast）
  • 鲁棒（Solid and Robust）
  • 简洁（Maintainable and Simple）
  • 共享（Re-usable）
  • 可测试（Testable）
  • 可移植（Portable）
  • 可监控（Monitorable）
  • 可运维（Operational）
  • 可扩展（Scalable&Extensible）
