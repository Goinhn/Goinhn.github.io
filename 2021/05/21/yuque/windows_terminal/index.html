创建文件

-file 为文件类型

-force 键入现有的文件全部更新，为空文件

-value 输入文件内容

new-item filename.type -type file -force -value “this is file content”

获取文件 hash

certutil

文档

​https://docs.microsoft.com/en-us/windows-server/administration/windows-commands/certutil

certutil -hashfile filepath hashtype

get-filehash

文档

​https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.utility/get-filehash?view=powershell-7

get-filehash filepath -algorithm hashtype|format-list
// 如果校验的文件路径比较复杂，例如路径中包含空格、括号等特殊符号，需要在路径前后加上英文的双引号
// 不添加-algorithm hashtype 的话默认为 sha256 校验

系统

开启卓越模式
以管理员方式进入 powershell 终端
输入命令
powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61
开启卓越模式
进入设置中的系统-电源和睡眠-其它电源设置-在电池指示器上选择卓越模式
可以通过如下软件进行切换平衡模式和卓越模式，不必手动进行设置
PowerPlanSwitcher
系统编码修改
windows 默认的编码为 GBK，编号为 936
使用 chcp 命令查看终端的编码
修改当前窗口编码，chcp 65001，只针对当前的活动窗口有效
修改所有窗口编码

  • 打开注册表，HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Command Processor
  • 右键单击 Command Processor，新建字符串，名称为 autorun，内容为 chcp 65001windows 默认的编码方式最号不要修改
