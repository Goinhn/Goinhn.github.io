剑指 offer

准备好合适的问题问面试官
编写程序之前多做沟通，做一个整体的设计和规划，有助于编写高质量和高可读性的代码，自我进行 code review
​

面试分类：

  • 电话面试：
    • 将细节的东西描述出来，对于问题没有听清楚的要和面试官确认保证理解的一致性
  • 远程面试编程的习惯（远程共享面试注重面试者的编写习惯和代码调试能力）
    • 思考清楚再开始进行编码
    • 良好的代码命名和缩进对齐的习惯
    • 进行单元测试（TDD）
  • 现场面试
    • 规划好去公司的路线和时间，避免迟到
    • 准备得体的衣着
    • 注意邀请函里的面试须知
    • 准备反问的问题

​

面试环节：

  • 行为面试
    • 面试者进行简短的自我介绍，包括学习和工作经历，面试官如果对项目感兴趣会有针对性的提几个问题详细了解
  • 技术面试
  • 应聘者提问
